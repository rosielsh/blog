# React useState Hook

리액트에서 가장 기본적인 Hook인 `useState`는 함수형 컴포넌트에서 상태를 관리할 수 있게 해주는 Hook입니다.

## 기본 사용법

```javascript
import { useState } from "react";

function Counter() {
  const [count, setCount] = useState(0);

  return (
    <div>
      <p>현재 카운트: {count}</p>
      <button onClick={() => setCount(count + 1)}>증가</button>
    </div>
  );
}
```

## 함수형 업데이트

이전 상태를 기반으로 새 상태를 계산할 때는 함수형 업데이트를 사용하는 것이 좋습니다.

```javascript
// 이전 상태를 기반으로 업데이트
setCount((prevCount) => prevCount + 1);
```

## 객체 상태 관리

객체 형태의 상태를 관리할 때는 스프레드 연산자를 사용해 이전 상태를 복사한 후 원하는 값만 변경합니다.

```javascript
const [user, setUser] = useState({
  name: "홍길동",
  age: 30,
  email: "hong@example.com",
});

// 이메일만 업데이트
setUser((prevUser) => ({
  ...prevUser,
  email: "new-email@example.com",
}));
```

## 지연 초기화

무거운 계산을 통해 초기 상태를 설정해야 할 경우, 함수를 전달하면 컴포넌트가 마운트될 때 한 번만 실행됩니다.

```javascript
// 함수 형태로 초기값 설정 (지연 초기화)
const [state, setState] = useState(() => {
  const initialState = someExpensiveComputation();
  return initialState;
});
```

useState Hook을 효과적으로 사용하면 리액트 컴포넌트에서 상태 관리를 간결하게 할 수 있습니다.
